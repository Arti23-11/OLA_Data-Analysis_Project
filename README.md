# 🚖 OLA Data Analytics Project

## 📌 Overview

This project focuses on analyzing OLA ride data to extract meaningful insights and support data-driven decision-making. It demonstrates an end-to-end data analytics workflow including data cleaning, querying, visualization, and reporting.

The project highlights practical skills in Excel, SQL, and Power BI, making it a strong showcase for data analyst roles.

---

## 📊 Dataset

The dataset contains ride-related information such as:

* Booking details
* Pickup and drop locations
* Ride status (completed, canceled, etc.)
* Fare and payment details
* Customer and driver information


---

## 🛠️ Tools & Technologies

* **Excel** → Data cleaning and preprocessing
* **SQL (PostgreSQL / MySQL / SQL Server)** → Data querying and analysis
* **Power BI** → Data visualization and dashboard creation

---

## 🔄 Project Workflow

### 1. Data Cleaning (Excel)

* Removed duplicates and handled missing values
* Standardized column formats (dates, text, numbers)
* Created calculated fields where required

### 2. Data Analysis (SQL)

* Wrote queries to extract insights such as:

  * Total bookings and revenue
  * Ride completion and cancellation rates
  * Peak booking times
  * Top locations and customer trends

### 3. Data Visualization (Power BI)

* Imported cleaned data into Power BI
* Built interactive dashboards with filters and KPIs
* Visualized trends using charts, maps, and graphs

### 4. Reporting

* Summarized key insights from analysis
* Highlighted business trends and actionable findings

---

## 📈 Dashboard Highlights

The Power BI dashboard includes:

* Total Rides, Revenue, and Cancellation Rate KPIs
* Time-based analysis (daily/weekly trends)
* Location-wise ride distribution
* Customer and driver insights
* Interactive filters for deeper exploration

---

## ✅ Results & Insights

* Identified peak hours for ride demand
* Analyzed cancellation patterns and possible causes
* Discovered high-performing locations and revenue drivers
* Provided insights to improve operational efficiency

---


## 📎 Conclusion

This project demonstrates a complete data analytics pipeline—from raw data to actionable insights—using industry-standard tools. It reflects strong analytical thinking, technical skills, and the ability to communicate data effectively.
